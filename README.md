## English

A Tampermonkey script that converts Katakana loanwords back to their original form.

### Features
- Auto-detects and translates Katakana words on any webpage
- Supports Google Translate and DeepL APIs
- Displays translations as Ruby annotations
- Customizable settings panel

### Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. [Install Rekata](https://github.com/PythonShe/Rekata/raw/main/rekata.user.js)
3. Configure your API key in the settings menu

### Usage
- Click the Tampermonkey icon → Rekata Settings
- Toggle translation on/off with the floating button
- Hover over Katakana words to see translations

## 简体中文

一个将片假名外来语转换回原始形式的 Tampermonkey 脚本。

### 功能特性
- 自动检测并翻译网页中的片假名词汇
- 支持 Google 翻译和 DeepL API
- 以振假名形式显示译文
- 可自定义设置面板

### 安装方法
1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. [安装 Rekata](https://github.com/PythonShe/Rekata/raw/main/rekata.user.js)
3. 在设置菜单中配置 API 密钥

### 使用说明
- 点击 Tampermonkey 图标 → Rekata 设置
- 使用浮动按钮开启/关闭翻译
- 鼠标悬停在片假名上查看译文

## 日本語

カタカナの外来語を元の形に戻す Tampermonkey スクリプトです。

### 機能
- ウェブページ上のカタカナ語を自動検出・翻訳
- Google 翻訳と DeepL API に対応
- ルビ形式で翻訳を表示
- カスタマイズ可能な設定パネル

### インストール
1. [Tampermonkey](https://www.tampermonkey.net/) をインストール
2. [Rekata をインストール](https://github.com/PythonShe/Rekata/raw/main/rekata.user.js)
3. 設定メニューで API キーを設定

### 使い方
- Tampermonkey アイコン → Rekata 設定をクリック
- フローティングボタンで翻訳の ON/OFF を切り替え
- カタカナ語にマウスを合わせると翻訳が表示されます
